В данном репозитории находится проект, реализующий тестовое задание для позиции PHP developer trainee.

Для реализации проекта были использованы следующие технологии и стандарты кодирования:

Backend:
PHP (стандарт кодирования: https://symfony.com/doc/current/contributing/code/standards.html )

Frontend:
jQuery (стандарт кодирования: https://contribute.jquery.org/style-guide/js/ )
HTML/CSS (стандарт кодирования: https://google.github.io/styleguide/htmlcssguide.html )

Согласно с поставленной задачей, реализованная веб-страница дает возможность получить инормацию о текущем курсе валют из 3х предложенных источников.

Источник данных передается файлу loadrates.php как аргумент source посредством POST-запроса. Далее этот аргумент передается в статическую функцию getData класса RateInformation.

В случае выбора локального файла в кач-ве источника, данные считываются из файла rates.dat. В кач-ве стороннего API для получения курса валют используется API Приватбанка.
